#  Freelance Web Development Portfolio

## About

My work focuses on minimalist web designs that direct the viewer's attention to the product rather than the web design itself. Anyone can make a really catchy webpage with smooth & attractive animations, but that is not the goal. From a business point of view a catchy web page directs that viewer's attention away from the advertising product itself and their eyes would wander everywhere throughout the screen wondering what to click next. A good webpage should make the viewer's task seem effortless and help direct their attention more on the product.

## 🛠 Tech Stack
- **Frontend:** Next.js, React, HTML5, CSS3, Tailwind CSS
- **Backend:** Node.js, FastAPI, Python
- **Database:** PostgreSQL (Supabase), SQL
- **AI & Integrations:** Claude, ChatGPT
- **Deployment & Tooling:** Git, GitHub Pages, Vercel, Docker, SquareSpace
